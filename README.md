# Pages & Parchment

Pages & Parchment is a digital reading room platform where readers gather to discuss books, share insights, and participate in focused literary conversations. The application provides dedicated discussion spaces for individual books with threaded comments, real-time chat, and ambient reading environments.

# Features
Book Discussion Rooms — Each book has a dedicated main room with topic-based subrooms for focused conversation
Quiet Reading Room — Immersive full-screen environment with ambient backgrounds, soft music player, and live whispers
Notifications — Real-time alerts for comment replies and room activity
Music Player — Curated playlists categorized by genre (Classical, Anime, Movie Soundtracks, Ambient, R&B)
User Authentication — Secure email/password authentication with JWT
Mobile Responsive — Fully functional across desktop, tablet, and mobile devices
